# NFC Service

01.12.2022 A new development

The innovative feature that turns the smartphone into a clearing terminal.  
Receiving payments by linking on the mobile from credit cards  
and digital wallets

Unlike manually typed transactions, the touch solution supports a secure transaction and allows you to pay with peace of mind even from the mobile device:  
Protection against transaction denials  
Clearing with EMV technology that has the strictest security standard

Using the API you can make a call for transmission to the device,  
And so turn your mobile device into a clearing device
