# Postman Collection

The Grow Postman Collection gives you one organized place to explore the full Grow API payment environment.  
It brings together key payment requests, response examples, sandbox setup, and relevant parameters for faster onboarding.

Sandbox credentials make it possible to simulate real payment activity before moving to live transactions.

This overview is especially useful for speeding up integration planning and reducing uncertainty across Grow payment flows.

## What is Postman?

Postman is a powerful, free-to-download tool designed for crafting and executing HTTP requests. Whether you are a developer, tester, or API enthusiast, Postman simplifies the process of working with APIs.
