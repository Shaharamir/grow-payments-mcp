# SetAccountDetails 

> This API allows you to define default branding settings for the business, such as logo, background color, and button styling. These settings will be automatically applied to all payment links generated by the business.

## `POST /api/light/server/1.0/SetAccountDetails`

**Servers:**
- http://sandboxapi.grow.link

**Summary:** SetAccountDetails Copy

**Parameters:**

| Name | In | Type | Required | Description |
|---|---|---|---|---|
| x-api-key | header | string | False |  |

**Request body** (`multipart/form-data`):

| Field | Type | Required | Description |
|---|---|---|---|
| userId | string | False | Required field- Unique identifier refers to every business that is connected and uses Grow payments solutions.   |
| pageCode | string | False | Required field -Unique identifier refers to payment mathod  |
| backgroundColor | string | False | Optional - Defines the background color of the payment page.  Format: HEX color code (e.g. #FFFFFF)  You can also configure a default business logo via the business management interface. |
| buttonColor | string | False | Optional - Defines the background color of the payment button.  Format: HEX color code (e.g. #0285AC)  You can also configure a default business logo via the business management interface. |
| paymentButtonText | string | False | Optional - Sets the text that will appear on the payment button  Validation rules: Up to 20 characters - only letters  You can also configure a default business logo via the business management interface. |
| businessLogo | string | False | Required field – Defines the business logo displayed in the payment link.  Provide a valid image URL that will be used as the default logo for the business.  Validations:  Maximum file size: up to 512KB Minimum resolution: 128x128 pixels Supported formats: square, rectangle, or circular images  The configured logo will be automatically applied to all payment links created by the business. |

**Responses:**

- `200` — Successful response
